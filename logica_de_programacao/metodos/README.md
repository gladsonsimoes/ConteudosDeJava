### como usar um metodo ?
### para invocar :
~~~java
          imprimirTraco();
~~~
#### final da linha tem que criar um metodo assim :
~~~java
        static void imprimirTraco() {
        String padraoTraco = "------------------------------------------------";
        System.out.println(padraoTraco);
    }
~~~
